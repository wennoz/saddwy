Switch: Variable

let nombre = "Esteban"
var edad = 19
nombre = "Jaider"
edad = 21
print("\(nombre) tiene \(edad) años de edad.")

¿Qué imprimía la última línea de código?

Toda aplicación eventualmente necesitara almacenar datos y Swift ofrece dos formas de hacerlo:

var: Una variable almacena datos que pueden cambiar cuando sea necesario.
let: Una constante almacena datos que no cambian una vez que han sido almacenados.
