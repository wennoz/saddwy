Las variables son contenedores para almacenar datos y en JavaScript se pueden declarar de 4 formas:

automáticamente: Se declaran automáticamente cuando se utilizan por primera vez.
var: Úselo únicamente si DEBE admitir navegadores antiguos.
let: Uselo para declarar una variable que puede ser reasignada.
const: Úselo siempre si no se debe cambiar el tipo de dato.
